class pynet.model.common.Logger(filelog=True, logfile='PyNet.log', file_log_level=LoggerLevel.DEBUG, consolelog=True, console_log_level=LoggerLevel.DEBUG)

Logger Singleton Class

Parameters

  • filelog (bool) – Flag activate file logging, defaults to True

  • logfile (str) – Filename to store the log, defaults to DEFAULT_FILE_NAME

  • file_log_level (LoggerLevel) – File log level, defaults to DEFAULT_FILE_LEVEL

  • consolelog (bool) – Flag activate console log, defaults to True

  • console_log_level (LoggerLevel) – Console log level, defaults to DEFAULT_CONSOLE_LEVEL

Examples

>>> logger = Logger()
